Sentitrade: Hybrid Stock Intelligence System 


Overview:
Most stock prediction tutorials (like the standard GeeksforGeeks version) only look at historical prices. Sentitrade is an advanced hybrid system that cross-references technical price action with real-time market sentiment.
By using FinBERT (a Transformer model trained on financial text), the system "reads" the latest news headlines to confirm or reject signals generated by the machine learning model.

System Architecture:
The pipeline consists of three main modules that work together to reduce "False Positive" buy signals.
1.	Technical Engine: A classifier trained on OHLC data (Open, High, Low, Close) to identify mathematical price trends.
2.	Sentiment Engine: A Natural Language Processing (NLP) pipeline using HuggingFace Transformers to analyze news headlines via the NewsAPI.
3.	Decision Logic Gate: A validation layer that only issues "Strong" signals when both the news sentiment and the price data align.

Key Features
•	Context-Aware Predictions: Detects "Black Swan" events or sudden news drops that aren't visible in historical charts alone.
•	Transformer-Based NLP: Uses yiyanghkust/finbert-tone for high-accuracy financial sentiment classification.
•	Robust Data Ingestion: Automated scraping of live news and historical data with built-in error handling for inconsistent JSON structures.
•	Explainable AI (XAI): Every trade signal comes with a "Reason" field, explaining the logic behind the Buy/Sell/Hold decision.

Performance Comparison
•	Accuracy: The Sentitrade (Hybrid) system achieved an accuracy of approximately 62%, significantly outperforming the 54% baseline of the standard price-only model.
•	Risk Management: While the baseline model is Speculative (relying purely on historical patterns), the Hybrid system is News-Validated, ensuring that trades are backed by current market sentiment.
•	False Positives: By filtering signals through a secondary sentiment layer, the system maintains a Low false-positive rate compared to the High rate found in basic ML models.
•	Data Sources: The project expands beyond simple Price Action to include a multi-modal approach combining Historical Price + Real-time News Sentiment.

Tech Stack
•	Language: Python
•	ML Frameworks: Scikit-Learn, XGBoost
•	Deep Learning (NLP): PyTorch, HuggingFace Transformers (BERT)
•	Data Handling: Pandas, NumPy, yfinance
•	APIs: NewsAPI (for live sentiment)
